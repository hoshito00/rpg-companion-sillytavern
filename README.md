# hoshito's RPG Companion — SillyTavern Fork

a fork of [SpicyMarinara's RPG Companion](https://github.com/SpicyMarinara/rpg-companion-sillytavern), rebuilt for more complex roleplay. i prefer stat-robust RPG systems, and this reflects that.

---

## what's new

### the stat sheet

a full character sheet with five tabs: attributes, jobs & feats, gear, augments, and combat skills. numeric stats, letter ranks, skill trees, gear with stat bonuses, a body-slot augment system, and a combat skill deck builder.

there's also a sixth tab, cultivation, still in progress.

### the combat engine

local dice resolution for turn-based encounters. the math happens client-side — damage, clashing, stagger, affinities, speed order — instead of being left to the AI. built around depth and chance, designed to feel kinetic rather than transactional. adapted from Project Moon's games and [Jakkafang's Stars of the City](https://docs.google.com/document/d/1BnU-VNWkLPjhtYfSfpaErkzdUd_LYk2deGrSQgsatXk/edit?tab=t.zgcrpcvhyh3f).
